Login Using ASP.NET

Explanation
I have built a login form using the Microsoft provided ASP.NET Configuration. Initially I have created a login form.The user needs to sign in using his credentials.If the User is new to the webpage he will have to Register.
The user in the Registration page has to enter all his details along with a security question and answer which will retrive his information in fututre, if he forgets his credentials.
The signed in user will be directed to the Member pages.He can then Change his password or go to member page. I havent given much functionality.I have just maintained a rough idea of a login page.
I have added a importatnt functionality wherein a User who has forgotten his password clicks on forget password in the login page gets password via an email from comicwebies@gmail.com
Also if the user wishes to remember himself then he can do so by clicking on remmember me on Login Page.
Here the change in web.config file for sending emails. 


      <mailSettings>
            <smtp from="comicwebies@gmail.com">
                <network host="smtp.gmail.com" password="webdevelopment" port="587"
                    userName="comicwebies" enableSsl="true" />
            </smtp>
        </mailSettings>
