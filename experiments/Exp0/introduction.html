Hover Effect on Images

I have implemented Prof.Rasala Image Hover Effect.
In this Experiment I when the mouse hover's over the static visible image it enlarges to a different Inage.
Application:The Image effect on hover can be used in places wherein there is not much space to display two Images. The following is the Javascript and CSS which are used to Implement the hover effect.

Javascript


		   function bigImg(image) {
						document.getElementById("normal").style.display = "none";
						document.getElementById("large").style.display = "block";
								   }
			function normalImg(image) {
						document.getElementById("normal").style.display = "block";
						document.getElementById("large").style.display = "none";
								}
