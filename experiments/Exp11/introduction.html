Movie Poster using CSS3

Explanation
I wanted to build something on movies which I might take up as my final project.I wanted to populate the movies using some 3d effect.
I did so by using webkit tansition and perspective property.On clicking the link the user is directed to the corresponding imdb movie


#movieposters { list-style:none; margin:100px 0; height:550px; }
#movieposters li { display:inline; float:left;
	-webkit-perspective: 500; -webkit-transform-style: preserve-3d;
	-webkit-transition-property: perspective; -webkit-transition-duration: 0.5s; }
#movieposters li:hover { -webkit-perspective: 5000; }
